## Security tooling for AI agents

Firmis Labs builds open-source tools to secure AI agent ecosystems — MCP servers, Claude skills, Codex plugins, and more.

### Projects

| | |
|---|---|
| **[firmis-scanner](https://github.com/firmislabs/firmis-scanner)** | Security scanner for AI agents. 227 rules, 9 platforms, 17 threat categories. |
| **[ai-inventory](https://github.com/firmislabs/ai-inventory)** | Detect AI frameworks, LLM dependencies, and model files in any project. |

### Quick start

```bash
# Scan any AI agent project for security threats
npx firmis-scanner scan .

# Inventory AI components in a project
npx @firmis/ai-inventory .
```
